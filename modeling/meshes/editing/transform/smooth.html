平滑¶

平滑顶点¶

参考

模式:编辑模式
面板:Toolbar ‣ Tools ‣ Mesh Tools ‣ Deform: Smooth Vertex
菜单:网格 ‣ 顶点 ‣ 平滑顶点

This tool smooths the selected vertices by averaging the angles between the faces. After using the tool, options appear in the Toolbar:

平滑
平滑系数。
重复
平滑的迭代次数。
轴
限制为仅作用于特定轴向。

平滑前的网格。

一次平滑后的网格。

10次平滑迭代后的网格。

See also

细分

在使用 细分 工具后调整 光滑度 选项可以获得更加圆润的形状。

See also

平滑修改器

平滑修改器 是另一种非破坏性的平滑工具，可以限制仅作用于单个 顶点组。

拉普拉斯平滑¶

参考

模式:编辑模式
菜单:专用项 ‣ 拉普拉斯平滑

详情可参考 拉普拉斯平滑修改器。

拉普拉斯平滑使用另一种平滑算法，可以更好地保留大的细节和网格的大致形状。拉普拉斯平滑除网格操作外，还有对应的非破坏性修改器。

Note

真实平滑 VS 着色平滑

不要将此工具与 这个页面 提到的着色平滑选项混淆，二者的原理并不相同！此工具通过修改网格本身来降低其尖锐程度，然而 设置光滑 / 自动光滑 只是通过控制网格的着色方式，来产生柔和的 错觉，但其实并没有修改网格。
