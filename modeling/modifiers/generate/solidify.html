实体化修改器¶

实体化修改器获取任意网格的表面，然后为之添加深度。

选项¶

实体化修改器.¶

厚度

待加固的深度。

偏移量

(-1 到 1) 之间的值用以定位实体化输出在原网格的外侧还是内侧。设为 0.0 的话，实体化输出在原网格面的中央。

限定

(0到2) 之间的值用以限制偏移量，可以避免自相交。

限定偏移量.¶

顶点组

仅该组的顶点进行实体化.权重系数会乘上厚度，这样低权重的顶点会更薄。

反转

反转顶点组，让 不 属于顶点组的顶点加固。

系数

顶点权重

  • 取 0.0 的话，权重为 0 的顶点将没有厚度。

  • 取 0.5 的话，权重为 0 的顶点的厚度是那些最大权重顶点厚度的一半。

  • 取 1.0 的话，权重被忽略，每个顶点都会使用 厚度 值。

折痕

这些选项是为了给同时使用 表面细分修改器 的.

边沿和棱。在这个例子里，物体被指定第二个之前用来给边沿上红色的材质¶

内

为内部边设置折痕。

外表面

为外部边设置折痕。

框

为边沿边设置折痕。

翻转法线

翻转所有几何形体的法线（包括内部和外部表面）。

均衡厚度

通过调整维持尖角的厚度。有时改善质量,但也会增加计算时间。

高品质法线

通过计算法线来获取更均匀的厚度。这样做有时可以提升质量，但也增加了计算时间。

填充边缘

填充内部和外部边之间的间隙。

仅边沿

不会有与原始平行的挤出表面，而是只有垂直边。

Note

填充边沿 和 仅边沿 仅在 非流形 物体上有区别，因为 边沿 是从原几何形体的边界产生的。

材质索引偏移量

对新的几何体选择不同的材质使用；这作为加固面原材质的偏移量而使用。

  • 值为 0 则使用相同的材质。

  • 值为 1 则使用原始材料正下方的材质。

  • 值为 -2 则使用原始材料上方两个单位的材质。

它们夹钳到最上和最下材质的材质槽里。

框

同样的，你也可以给另一个材质到边沿的面上。

Important

修改器的厚度是用局部顶点坐标计算的。如果物体用了非均匀缩放，厚度在物体不同的边上会有所变化。

为了修复这个问题，需要应用 Ctrl-A 或 清除 Alt-S 缩放。

已知局限¶

均衡厚度¶

实体化厚度是近似值。 虽然 均匀厚度 和 高质量法线 应该产生良好的结果，但最终的壁厚不能得到保证，并且可能根据网格拓扑而变化。

为了在每种情况下都能维持精确的壁厚，可能需要添加或移除偏移外壳上的面,一些这个修改器不会做的事，因为这样会添加很多复杂性，让修改器变慢。
