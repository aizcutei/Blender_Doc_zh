添加 ‣ 文本

文本 物体包含文本。由于现代字体是矢量化的，由曲线构成，因此其与 曲线 和 曲面 为同一物体类型。

Blender使用一套 "字体" 来管理文字符号到3D视图中物体的映射关系。该字体系统有自己 内建 的字体，但也可以使用外部字体，如 PostScript Type 1, OpenType 与 TrueType 字体。另外，它可以使用当前blend文件中的任何物体表示字符。

例: 挤出后的字体。¶

文本 物体用于使用各种布局选项如排版和框架，来创建和渲染二维或三维文本。字符默认为平面填充的曲面，和闭合二维曲线一样。但是，跟 曲线 一样，可以对其进行挤出。文本还可设置为跟随曲线。

Tip

可以在 物体模式 下使用 转换为 将 文本 转换为曲线，或者直接转换为网格。

Note

单个文本物体允许使用最多50000个字符；不过事先警告的是，单个文本物体使用的字符越多，物体的交互响应会变得更慢。

Note

在 物体模式 和 编辑模式 之间切换时， 字体 面板维持不变。这意味着可以在两种模式下同样地应用这些设置，也意味着这些设置无法单独作用于文本的一部分。所以同一个 文本 物体中字符的字体、尺寸等是相同的。只有一个例外： 粗体 或 斜体 按钮控制的属性可以指定至单个字符(这是在同一文体中使用最多4种字体的方法)。
