选择 & 编辑¶

在 Blender 中编辑文本与别的物体相当不同, 主要是在两个位置. 首先, 在3D 视图中输入你的文本, 这里有几个快捷键, 例如应用样式 (见 字符) -- 但要注意, 大多数你所知道的 Blender 编辑模式 的热键都不适用于文本. 第二个地方时属性编辑器, 特别是 字体 标签.

选择和游标¶

编辑模式中的文本.¶

在 编辑模式 中, 你的文本有一个白色的光标, 就像任何文本编辑器那样, 它确定了字符被插入的位置.

下一个/上一个字符

你可以移动光标, 通过方向键 Left 或 Right.

下一个/上一个词

为了在词的边界间移动光标, 使用 Ctrl-Left 或 Ctrl-Right.

行首/行尾

Home 和 End 分别能移动光标到行的开始和结尾.

下一行/上一行

为了在行间跳转, 使用 Up 或 Down.

下一页/上一页

为了一次向前或向后跳转10行, 使用 PageUp 或 PageDown.

在使用方向键的同时按住 Shift 可以选择一部分文本. 你可以用它来指定不同的材质, 正常/粗体/斜体样式, 以及不太多的别的属性.

编辑¶

3D 视图头的菜单提供了几个选项, 在这里没有 专用项 菜单. 你既没有变换也没有镜像等等. 但是, 你可以将和曲线的一样修改器应用到文本上.

基本¶

编辑 文笔 与使用标准文本编辑器相似, 但是不具有全部的功能, 而且还有一些区别:

退出 编辑模式

Tab 不会在文本中插入一个制表符, 而是进入和退出 编辑模式, 就像别的物体类型那样.

复制

为了将文本复制进缓冲区, 使用 Ctrl-C 或者工具栏中的 复制 按钮.

剪切和复制

为了剪切和复制文本进缓冲区, 使用 Ctrl-X 或工具栏中的 剪切 按钮.

粘贴

为了将缓冲区中的文本粘贴出来, 使用 Ctrl-V 或者工具栏中的 粘贴 按钮.

删除所有文本

为了完全清除或删除所有文本, 使用 Ctrl-Backspace.

文本缓冲区与桌面剪贴板是同步的. 但是如果它是在 Blender 内部使用, 文本格式也会一起复制. 对于其它的插入文本方式, 见后文的 Inserting Text .

特殊字符¶

文本 ‣ 特殊字符

如果你需要特殊字符 (例如音标字符, 这种不在你的键盘上的) 你可以将两个其它字符组合起来用以制作很多特殊字符. 为此, 输入主要的字符, 按下 Alt-Backspace, 然后按想要的 "修改器" 来制作特殊字符. 下面给出了一些例子:

ã

A, Alt-Backspace, ~

á

A, Alt-Backspace, '

à

A, Alt-Backspace, \

å

A, Alt-Backspace, O

ë

E, Alt-Backspace, "

ø

O, Alt-Backspace, /

插入文本¶

你可以通过两种方式插入文本: 从内部文本缓冲区 (如上文所述), 或者从文本文件中.

为了从文本文件中载入文本, 使用 文本 ‣ 粘贴文件 工具. 这会打开一个 文件浏览器 用来定位一个有效的UTF-8文件. 如其它一样, 要当心文本不要有太多字符, 因为交互响应会变慢.

转换文本物体¶

转换成文本物体¶

在文本编辑器标题栏选择 编辑 ‣ 将文本转换为 3D 物体，根据需要选择 一个物体 或 每行生成一个物体，可以将现有的文本数据块转换为物体。

在编辑3D文本时, 也可以从剪贴板粘贴文本或者在 编辑 菜单里选择从文件粘贴.

转换成3D网格¶

It is possible to convert a Text Object to a 3D Mesh object. This can be useful so that you may edit the vertices in Edit Mode, but you will lose the ability to edit the text itself.

This can be done, using 转换为 in Object Mode. Now you can return to Edit Mode and manually edit the vertices.

Tip

The result is usually a bit messy, so it may be useful to use a Limited Dissolve deletion or Remesh Object Modifier at a low threshold to clean up your mesh.

指定材质¶

属性编辑器 ‣ 材质

要使不同的字符获得不同的材质，每个字符都必须有一个不同的 材质编号 。

你可以通过输入的方式制定材质编号，或先选择文本块，再点击材质面板上的 指定 按钮。

红 绿 蓝。¶

例如, 为了创建图 红 绿 蓝。 你需要分别创建三个材质和三个材质索引. 通过选择每个词的字符并点击 指定 按钮, 每个词会被分派一个 材质索引. 图 红 绿 蓝。 仍然是一个单个的 文本 物体.
