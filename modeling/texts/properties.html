载入和改变字体¶

Blender 默认带有 内置 字体, 这显示在每个四个字体样式数据块菜单里. 内置 字体总是在这个列表里, 并显示为 "Bfont". 数据块菜单包括一个列表, 其中显示着当前载入的字体. 为每个字体样式选一个字体.

为了载入一个不同的 字体 , 点击 字体 面板中的 载入 按钮中的一个, 并且定位到一个字体文件. File Browser 会给出所有有效字体一个大写 "F" 图标.

如果你选择一个 Blender 不支持的字体, 会显示错误 "不是一个有效字体".

Note

Unix 中的字体位置

通常字体在 /usr/lib/fonts 里, 或者类似的路径 /usr/lib/X11/fonts, 但也不总是这样. 也有可能在别的位置, 例如 /usr/share/local 或 /usr/local/share, 或在可能的相关子树里.

记住, 相同字体会被应用在一个文本物体的相同样式的所有字符上, 但每个样式都需要分别设置字体. 例如, 你会需要载入一个 斜体 字体, 为了让字符或文字是斜体. 一旦字体被载入, 你可以应用这个字体样式到选定的字符或整个物体. 总共你要加载最少四种不同类型的字体, 来表示每种样式(正常,斜体, 粗体, 粗斜体).

重要的是要明白，Blender 不在乎你对 "正常" "粗体" 等样式载入了什么字体. 这也就是说你可以在同一个文本中有四个不同的字体, 但是你必须选择是同一个字体的不同样式或者是不同的字体. Blender有很多印刷控制, 用来改变文本的样式和布局, 这些都可以在 字体 面板中找到.

尺寸和切变¶

尺寸

控制整个文本的尺寸(不能单独控制每个字符). 注意: 不同字体的字符(不同样式, 见下文) 可以看起来的大小不同.

切变

控制整个文本的倾斜度. 与它看起来不同, 这与斜体样式不一样.

切变的例子.¶

物体作为字体¶

在 Blender 里, 你也可以 "创建" 你自己的 "字体"!这是相当复杂的过程, 所以我们会详细说明:

  1. 首先, 你必须创建你的字符. 任何类型的每个字符都是一个物体(网格, 曲线, 元物体/融球...). 他们全都必须有一个遵循下面结构的名字: 共同前缀 后面紧跟着 字符名字 (例如 "ft.a", "ft.b" 等).

  2. 接下来, 对于 文本 物体, 你必须启用 副本顶点 按钮 (物体 ‣ 动画设置 面板).

  3. 在 字体 标签里, 将你的"字体"物体的 共同前缀 填入 物体字体 区域里.

现在, 每一次你的文本中的一个字符与"字体"物体的名字的 后缀部分 相符合, 这个物体就会被复制到这个字符上. 原来的字符仍然可见。物体被复制以便于他们的中心位于响应字符的 右下角.

曲线上的文本

用来选择一个曲线, 来让文本物体跟着它.

这控制下划线的粗细.

字符¶

字符选项来输入(例如)粗体文本.¶

粗体

在输入前切换 粗体 按钮. 文本也可以被设为粗体, 通过先选中它然后用工具栏中的 粗体 按钮.

斜体

在输入前切换 斜体 按钮. 文本也可以被设为斜体, 通过先选中它然后用工具栏中的 斜体 按钮.

下划线

启用下划线, 就像由上面的下划线设置所控制的那样.

小型大写

输入小的大写文本.

Blender的 粗体 和 斜体 按钮与别的软件工作方式不一样, 因为他们也作为你手动载入其它字体的占位符, 这用在你定义相应样式时, 见 Font.

为了应用粗体/斜体/下划线属性到一系列字符上, 你要么在输入字符前打开 粗体 / 斜体 / 下划线 选项, 要么先高亮(选中)然后切换粗体/斜体/下划线按钮.

设置大小写¶

你可以改变文本的大小写, 通过选中它然后点击工具栏中的 转大写 或 转小写.

启用 小型大写 选项来元如小型大写的字符.

小型大写 的大小可以由 小型大写缩放 设置项来改变. 注意, 小型大写缩放 被用在所有 小型大写 格式的字符上.

段落面板.¶

水平对齐¶

左

使用时对齐文本到框架的左侧, 或者使用 文本 物体的中心点作为文本的起点(之后延伸到右侧).

中心

使用时在框架中居中对齐文本, 或者使用 文本 物体的中心点作为文本的中点(之后向左右两侧等长延伸).

右

使用时对齐文本到框架的右侧, 或者使用 文本 物体的中心点作为文本的终点(之后延伸到左侧).

左右对齐

只有被自动换行终结时才会换行 (不会not 被 Return 换行), 它用 空白 而不是 空格字符 (紧排地) 填充一行.

分散对齐

总是换行, 即使是还在输入时, 它使用空格字符(紧排地)填充一行.

左右对齐 和 分散对齐 只在框架内可用.

垂直对齐¶

顶基线

使用时对齐文本基线到框架的顶部, 或者使用 文本 物体的中心点作为文本的起点(之后延伸到底部).

顶

使用时对齐文本顶部到 文本 物体的中心点(之后延伸到底部). 当使用框架时, 它的行为和 顶基线 相同. 只在没有框架时 顶 才可用

中心

使用时在框架中居中对齐文本, 或者使用 文本 物体的中心点作为文本的中点(之后向上下两侧等长延伸).

底

使用时对齐文本到框架的底部, 或者使用 文本 物体的中心点作为文本的终点(之后延伸到顶部).

间隔¶

字符

每个字符之间的空间按比例缩放的一个因子。

词

每个单词之间的空间按比例缩放的一个因子。你可以控制它, 通过按下 Alt-Left 或 Alt-Right 来减少/增加间距, 每次0.1.

行

每行之间的垂直空间按比例缩放的一个因子。

偏移量¶

X 向偏移和 Y 向偏移

这些设置项控制文本的 X 和 Y 方向偏移, 相对于它 "正常" 的位置. 注意, 文本框, 它被用在所有框架的内容中.

文本框¶

参考

模式

所有模式

面板

属性编辑器 ‣ 字体 ‣ 文本框

文本“框”允许您将文本分布在单个文本对象内的矩形区域中. 每个文本物体允许有任意数量的可自由定位和可重缩放的文本框架.

文本从最低编号的框架连续地流到最高编号的框架, 每个框架中的文本都可换行. 当较低编号的框架不能再多放任何文本时, 文本在框架之间流动. 如果最后的框架都到了, 文本会溢出来.

文本框面板.¶

边界尺寸

指定框架的 宽度 和 高度 . 如果设成 0 不会自动换行.

偏移量

控制框架的 X 和 Y 向偏移.

添加文本框

在当前框架后插入一个新框架(按文本流动的顺序). 新框架会与选中的那个有相同的大小和位置.

删除(X 图标)

删除当前框架.
