支持的视频格式¶

视频格式¶

这些格式主要用于将渲染完成的序列压缩成可播放的电影（它们也可以被用来制造普通的音频文件）。

编码器是压制视频的常用工具，使视频适用于DVD或者通过互联网、缆线的串流播出，或只是为了获得合适的文件大小。编码器把视频的各个通道压制进储存空间并支持连续播放。有损编码器减小文件尺寸是以图像品质的损失为代价。一些编码器，如H.264，适用于较大的图像。编码器可用来编码和解码视频，所以其同时存在与编码机器(Blender)和目标机器。编码后的数据储存在文件中。

行业内有几十个，甚至几百个，编解码器，包括XviD格式，H.264，DIVX，微软，等等。每个都有其优点和缺点，并兼容不同的操作系统不同的播放器。

Note

大多数编解码器只能压缩RGB或YUV 色彩空间，但一些编码器也很好地支持Alpha通道。支持RGBA的编解码器包括：

  • QuickTime

  • PNG TIFF Pixlet 并非无损，并且可能是仅适用于macOS。

  • Lagarith无损视频编解码器

AVI编码器

AVI编码压缩。可用编码器与操作系统相关。选择AVI编解码器后，编解码对话框会自动启动。该编解码器可以使用出现的 设置编解码器 按钮进行更改（AVI解码器设置）。

AVI Jpeg

使用JPEG压缩的AVI。有损，能得到更小的文件，但大小无法与编解码器的压缩算法得到的文件相比。数字摄像机中使用的DV格式中也有使用JPEG压缩。

AVI Raw

音视频数据交叉(（AVI）的未压缩的帧集。

帧服务器

Blender将基于请求的帧作为渲染农场的一部分。端口号在 用户设置 中指定。

H.264

使用 H.264对电影进行编码。

MPEG

使用Xvid 编解码器对电影编码。

Ogg Theora

使用Theora 编解码器将电影编码为 Ogg 文件。

QuickTime

苹果的 Quicktime .mov 文件。在在macOS 安装该编解码器，可以使用Quicktime编码解码器对话框。 参考 视频格式 中的 Quicktime 。

Xvid

使用Xvid 编解码器对电影进行编码。

视频文件¶

MPEG-1: .mpg, .mpeg

视频和音频标准有损压缩格式。设计用于压缩VHS质量的原始数字视频和低至1.5兆比特/秒的CD音频。

MPEG-2: .dvd, .vob, .mpg, .mpeg

“运动图像及其相关音频信息通用编码” 的标准。它描述了有损视频压缩和有损音频压缩方法的合集，其允许存储和传输使用目前可用存储介质和传输带宽的电影。

MPEG-4(DivX): .mp4, .mpg, .mpeg

吸收了MPEG-1和MPEG-2以及其他相关标准的功能，并增加了新的功能。

AVI: .avi

资源交换文件格式(RIFF)衍生格式，它把文件的数据分成块，或 “chunk”。

Quicktime: .mov

一种多跟踪格式。 QuickTime和MP4文件格式可以使用相同的MPEG-4格式；它们在仅使用QuickTime的环境中通常可交换。 MP4作为国际标准，有更多的支持项。

DV: .dv

帧内视频压缩方案，它使用离散的余弦变换（DCT）来从帧接帧的基础上压缩视频。音频无压缩存储。

H.264: .avi for now.

一种视频压缩标准，是目前记录，压缩和分配高清晰度视频最常用的的格式之一。

Xvid: .avi for now

遵循MPEG-4标准的视频编解码器库。使用ASP功能如b帧，全局和四分之一像素运动补偿，亮度遮蔽，网格量化，和H.263，MPEG，自定义量化矩阵。 XVID是DivX Pro编解码器的主要竞争对手。

Ogg: .ogg, .ogv

自由的有损压缩格式。它是由Xiph.Org基金会开发并免费分发。

Matroska: .mkv

一个开放的标准自由文件格式，可以在单个文件中容纳无限数量的视频，音频，图片或字幕轨道的文件格式。

Flash: .flv

文件格式，用于提供使用AdobeFlash播放器的互联网视频服务。

Wav: .wav

未压缩(或轻度压缩)的微软和IBM音频文件格式。

Mp3: .mp3

高度压缩的，有专利的数字音频编码格式，采用有损数据压缩形式。它是一种常见的存储音频的格式，同时也是电子播放器上传输和播放音乐所用数字音频压缩格式的事实标准。

视频编码¶

无

只编码声音。

MPEG-1

见 视频格式.

MPEG-2

见 视频格式.

MPEG-4(DivX)

见 视频格式.

HuffYUV

Ben Rudiak-Gould创造的无损视频编码，意在取代无压缩YCbCr作为视频截图格式。

DV

见 视频格式.

H.264

见 视频格式.

Xvid

见 视频格式.

Theora

见 视频格式 中的Ogg。

Flash 视频

见 视频格式.

FFmpeg视频编码 #1

也称作 FFV1，一种无损帧内视频编码。可使用可变长度编码或算术编码用作熵编码。编码器与解码器是FFmpeg中 libavcodec的一部分。

音频文件¶

MP2

一种由ISO/IEC 11172-3制定的有损音频格式。

MP3

见 视频格式 中MP3部分。

AC3

Audio Codec 3, 一种Dolby实验室开发的音频格式。

AAC

高级音频编解码器，标准化的有损压缩和数字音频的编码方案。 -- 相似码率情况下，AAC音质通常比MP3好。

Vorbis

开放标准、高度压缩的格式，可媲美MP3或AAC。-- 相似码率情况下，Vorbis音质通常比MP3好。

FLAC

自由的无损音频格式，FLAC编码的音频文件大小通常可以降低至原始文件的50-60%，并且解压为与原始音频数据相当的副本。

PCM

Pulse Code Modulation，一种以数字方式表示模拟信号的方法，是电脑、蓝光光碟，CD，DVD格式中数字音频的标准形式，也有一些其他用途，如数字电话系统。

已知局限¶

视频输出尺寸¶

一些编码对输出尺寸有所限制，如 H.264 要求宽度和高度均能被2整除。
