雕刻笔刷。¶

球体

使用通过控制设置球体边的夹捏量的球形将网格向外或向内推。

Clay C 粘土 C

Similar to the SculptDraw brush, but includes settings to adjust the plane on which the brush acts. It behaves like a combination of the Flatten and SculptDraw brushes.

Clay Strips 粘土条

类似于 Clay（粘土） 笔刷，但它使用立方体测试来定义笔刷区域的影响而不是球体。

Crease Shift-C 折痕 Shift-C

通过推或拉动网格，同时将顶点挤压在一起，创建尖锐的凹痕或凸脊。

Fill/Deepen 填充/深化

Works like the Flatten brush, but only brings vertices below the brush plane upwards. The inverse of the Fill brush is to Deepen by pushing vertices below the plane downward.

Flatten/Contrast Shift-T 展平 Shift-T

The Flatten brush determines an "area plane" located by default at the average height above/below the vertices within the brush area. The vertices are then pulled towards this plane. The inverse of the Flatten brush is the Contrast brush which pushes vertices up or down away from the brush plane.

Grab G 移动

用于拖动一组顶点。与其它笔刷不同，Grab 不会在笔刷拖过模型时修改不同的顶点。相反，Grab 只会选择在鼠标按下上的一组顶点，它们将会跟随鼠标拉动。效果类似于使用衰减编辑功能在 编辑模式 中移动一组顶点，除外 Grab 可以使用其他雕刻模式选项（如纹理和对称性）。

Inflate/Deflate I 膨胀 I

与 SculptDraw 类似，除了 Inflate（膨胀） 模式中的顶点在自己的法线方向上移位。

Layer L 层次 L

这个笔刷类似于 SculptDraw，除了位移层的高度被叠加绘制。这将产生正在绘制的实心层的外观。这个笔刷不能在自己的顶部绘制; 笔刷笔触与本身相交。释放鼠标按钮并开始新的笔触将重置深度并在上一个笔触之上绘画。

Persistent

You can keep sculpting on the same layer between strokes when this is on.

Set Persistent Base

This button resets the base so that you can add another layer.

Mask 遮罩 M

让你通过绘制顶点颜色来选择不受其他画笔影响的部分网格。遮罩值显示为灰度。即一个遮罩面的区域越暗，雕刻的效果就越少。另请参阅 遮罩 菜单的选项。

遮罩工具

遮罩刷有两种模式：

绘画

Mask drawing. 遮罩绘画。

平滑 Shift

按 Shift 遮罩笔刷将切换到平滑模式。

Nudge 微调

沿着笔刷笔触的方向移动顶点。

Pinch/Magnify P 夹捏/放大 P

将顶点拉向笔刷的中心。反向设置为 Magnify，其顶点从笔刷的中心向外推开。

Rotate 旋转

沿着光标移动的方向旋转画笔中的顶点。初始拖动方向是零角度，通过围绕中心旋转可以创建涡流效果。

Scrape/Peaks 削刮

削刮 笔刷像展平笔刷一样，但只能将平面上方的顶点向下推。削刮的反向设置为 Peak 是通过将平面上方的顶点推离开平面。

SculptDraw 绘画笔刷 X

基于包含在绘制的笔触中的顶点的平均法线，向内或向外移动顶点。

平滑 S

顾名思义，通过平滑顶点的位置来消除因为笔刷影响的不规则性网格部分。

Snake Hook 钩子笔刷 K

沿着笔刷的运动反向拉起顶点以创建长的，类似蛇的大形。

Pinch 夹捏

The Snake Hook brush tends to loose volume along the stroke, with pinch > 0.5 it's possible to sculpt shapes without loosing volume.

Rake 变向

支持按照游标旋转运动来移动网格。

Thumb 拇指笔刷

类似于 Nudge微调 笔刷，这一个笔刷使笔刷区域中的网格平整，同时沿着笔刷的笔触移动。

Simplify

无论 折叠短边 选项是否启用，此笔刷将折叠短边（由细节大小定义）。如果未启用动态拓扑，则此刷无效。可以在 Brush ‣ Sculpt Tool 中找到菜单。

常用选项¶

Radius 半径

此选项以像素为单位控制笔刷的半径。F 允许你通过拖动鼠标交互式更改笔刷大小，然后按住 LMB （画笔的纹理应该在圆圈内可见）。键入数字，然后在使用时输入 F 允许你以数字形式输入大小。如果你使用数位板（ Graphics Tablet），则可以通过启用压力敏感图标来影响画笔大小。

Strength 强度

控制每个笔刷如何应用对模型的影响。例如，较高的值会导致 SculptDraw 笔刷更快地为模型添加深度，并使 Smooth 画笔更快地平滑模型。此设置不适用于 Grab，Snake Hook 或 Rotate。

你可以通过在3D视图中按下 Shift-F 来交互地更改笔刷强度，然后移动笔刷，然后 LMB 确认。你可以按住 Shift-F 下输入数值大小。如果使用支持的平板电脑，则可以通过启用压力敏感度图标来影响笔刷强度。

Tip

如果强度范围似乎不符合模型（例如，即使最低强度设置仍然使模型发生变化太大），则可以缩放模型（ 编辑模式，而不是 对象模式 ）。较大的尺寸将使刷子的效果更小，反之亦然。

Autosmooth 自动平滑

设置要应用于每个笔刷的平滑量。

Normal Weight 法线权重 Ctrl

沿着表面法线限制笔刷运动。特别适用于 Grab Brush，可以通过按住 Ctrl. 暂时启用。例如当设置 法线权重 时，Grab笔刷可以使用推动将凹陷（孔）推入网格。

适用于 Grab 和 Snake Hook 笔刷。

Use Original Normal (padlock icon) 使用原始法线（挂锁图标)

When locked it keeps using the normal of the surface where stroke was initiated, instead of the surface normal currently under the cursor.

Sculpt Plane 雕刻的平面

Use this menu to set the plane in which the sculpting takes place. In other words, the primary direction that the vertices will move.

Area Plane 区域面

平面位于笔刷区域内顶点的上方/下方的平均高度。本质上，这意味着方向取决于笔刷下方的表面。

View Plane 视图平面

Sculpting in the plane of the current 3D View.

X, Y, Z Plane X, Y, Z 平面

The movement takes place in the positive direction of one of the global axes.

Plane Offset 平面偏移

Offset for planar brushes (Clay, Fill, Flatten, Scrape), shifts the plane that is found by averaging the faces above or below.

Trim 裁剪

Ability to limit the distance that planar brushes act. If trim is enabled vertices that are further away from the offset plane than the trim distance are ignored during sculpting.

Front Faces Only 仅前面的面

启用时，画笔只影响面向视图的顶点。

2D Falloff

This turns the brush influence into a cylinder (the depth along the view is ignored) instead of a sphere. It can be used along the outline of a mesh to adjust its silhouette.

Add/Subtract Ctrl 添加、相减

Brush direction toggle. Ctrl pressed while sculpting. 当雕刻时按 Ctrl 笔刷方向切换。

Accumulate 累积

导致笔触在顶部彼此累积。

对称/锁定面板¶

参考

雕刻模式

面板

Tool Shelf ‣ Tools ‣ Symmetry/Lock

菜单

Sculpt ‣ Symmetry/Lock

Mirror 镜像

在所选局部轴上对笔刷笔触进行镜像。请注意，如果要更改轴指向的方向，则必须以 编辑模式 旋转模型，而不是 物体模式。

Radial 径向

这些设置允许所需轴线的径向对称。该数字确定在中心轴周围360度内重复行程的次数。

Feather 羽化

降低与对称平面重叠的笔触强度。

Lock 锁定

当你雕刻它时，这三个按钮允许你阻止模型沿着选定的局部轴的任何修改/变形。

Tiling 平铺

使用此选项可以让您沿给定的轴无缝地平铺你的笔画。这允许创建重复模式。

Tile Offset 平铺偏移

默认的平铺大小设置为 BU。偏移允许选项沿所有三个轴改变平铺大小。
