介绍¶

打开Blender并关闭了 启动画面 之后，你会看到与下图相似的Blender窗口。Blender的用户界面在所有平台上都是一致的。

默认的Blender启动界面。¶

界面元素¶

Blender界面分为以下四个界面元素：

  • 最顶部的顶部菜单。

  • 顶部菜单下方的工具设置。

  • 中间的编辑器区域。

  • 底部的状态条。

大的编辑器区域进一步分为包含不同编辑器的 区域。可以自定义用户界面为 工作空间 ，并命名和保存，以满足后续特定工作需要。布局空间将在下文介绍。

Blender默认屏幕布局。顶部菜单(蓝色)，工具设置(黄色)，编辑器(绿色)和状态条(红色)。¶

布局工作空间¶

Blender默认的启动文件显示的是名为 "Layout" 的工作空间。该工作空间包含以下 编辑器:

  • 3D视图位于左上。

  • 大纲视图位于右上角。

  • 属性编辑器位于右下角。

  • 时间轴位于左下方。

编辑器的组成部分¶

总的来说，一种编辑器提供了使用Blender查看与修改作品的一种方式。

编辑器由 区块 组成。而区块则由更小的结构元素构成，比如包含按钮，控制条和窗口小部件的 选项卡和面板 。

3D视图编辑器。¶

标题栏(绿色)，主区块(黄色)，工具栏(蓝色)，侧栏(红色)和操作面板(粉色)。

用户界面原则¶

没有重叠

UI的设计是为了让你能一目了然所有相关的选项和工具，而无需到处拖拽编辑器。

没有遮挡

工具和界面选项不会在用户使用任何Blender的其他功能时造成遮挡。Blender的特点是没有弹出式对话框（要求用户在进行操作之前输入数据）。

非模态工具

用户可以高效地访问工具而不需要花时间在不同工具之间选择。很多工具使用了一致的且可预知的鼠标和键盘操作来进行互动。

个性化定制¶

Blender也大量地使用了快捷键来加速工作。这些快捷键也可以在 键盘映射编辑器 中定制。

主题色彩

Blender允许用户按照需求来更改大多数用户界面的颜色。如果你发现屏幕上的颜色和手册里提到的那些不一样，那么有可能你的默认主题被人改了。选择 用户偏好 编辑器并点击 主题 选项，即可创建一个新的主题或选择/更改一个现有的主题。
