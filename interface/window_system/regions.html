区块-Regions¶

每个编辑器由几个区块组成。区块又可以划分为较小的结构元素如 选项卡&面板 ，其中包含按钮，控件和小部件等。

在添加一个立方体的后，3D视图编辑器会在侧栏区块显示调整上一步操作面板。

标题栏（绿色），主区块（黄色），工具栏（蓝色），侧栏（红色）和调整上一步操作面板（粉色）。

主区块¶

有一个区块是始终可见的，这也就是所谓的主区块，它是编辑器最重要的部分。

每个编辑器都有特定的用途，所以每个编辑器的主区块和附加区域的功能都是不一样的。更多信息见 编辑器 章节中各编辑器对应文档。

标题栏¶

标题栏是位于区域顶部或底部以浅灰色为背景的水平长条。所有编辑器的标题栏都作为菜单和常用工具的容器。 菜单 和按钮随编辑器类型以及选中对象及其模式而变化。

3D视图编辑标题栏。

标题栏菜单¶

RMB 单击在标题上显示一个带有几个选项的上下文菜单：

显示标题栏
切换显示标题栏，如果标题栏隐藏了，可以拖动编辑器上下边缘的右侧小箭头来重新显示标题栏。
显示菜单
切换显示 工具设置。
上下反转
切换 菜单 是否折叠。
上下反转
切换标题栏或工具设置显示在编辑器的上面还是下面。
区域最大化
见 切换区域最大化 。

工具栏¶

工具栏 (位于编辑器左侧)包含工具设定。 T 切换显示/隐藏工具栏。

进一步了解: 工具栏。

工具设置¶

工具设置 （在编辑器区域的上方或下方）包含当前活动工具的名字和设置。可以在标题栏内容菜单中的 反转 顶部/底部 改变位置。

调整上一步操作¶

调整上一步操作 是当工具（操作）运行时的设置选项区块。

进一步了解: 调整上一步操作 。

侧栏¶

侧栏 (位于编辑器右侧)，包含编辑器中对象与编辑器自身的设置 面板 。 N 用于显示/隐藏侧栏。

底栏¶

一些编辑器显示一个栏（在编辑器区域的顶部/底部），显示有关例如活动工具的信息。

布局¶

滚动¶

区块可以拖动 MMB 垂直/水平滚动，如果该区块没有缩放级别，鼠标停留在上方即可使用 Wheel 滚动。

改变尺寸与隐藏¶

与 区域-Areas 一样，可以使用拖拽边界调整区块大小。

调整区块的大小直至隐藏，隐藏的区块会留下一个箭头图标，用 LMB 点击这个图标，区块会重新显示。
