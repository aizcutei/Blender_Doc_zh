简介¶

文件浏览器。¶

用法¶

文件浏览器用于所有与文件相关的操作。它有许多用途，通常用于保存/加载。

包括：

  • 打开和保存Blend文件。

  • 导入/导出其他文件格式。

  • 为现有的文件路径（图像，视频，字体...）选择新的位置。

  • 当使用 关联库 时，浏览其它blend文件内部。

和其他任何编辑器类型一样，你也可以保持文件浏览器处于打开状态以浏览文件系统。在这种情况下，确认/取消按钮将不存在。

这样做的主要目的是为了能够拖动媒体文件：

  • 将图像放入 视频序列编辑器 （设置背景或应用为材质纹理）。

  • 媒体文件放入 视频序列编辑器。

另一方面，如果打开文件浏览器进行文件操作（打开，保存，导入等），它将显示最大化并等待操作完成，然后返回到之前的屏幕布局。

文件区域¶

文件路径

当前路径的文本字段。 Tab 会自动完成现有的路径。如果你输入一个不存在的目录路径，你会被提示创建新的目录。

文件名

文本字段-用于编辑文件名和扩展名。如果背景为红色，则文件夹中已存在具有相同名称的文件。 Tab 会自动完成当前目录中的现有名称。

递增文件名 +, -

用于添加/增加或删除/减少文件名后缀数字(用于表示文件的 版本 )。

确认

打开目录/文件的主要按钮或保存（另存为） Return 或双击 LMB 在条目上确认该文件或数据块。

  • Shift-LMB -- 从外部打开文件 (在 文件 中选择).

  • Alt-LMB -- 在外部打开目录 (使用系统的文件管理器)。

取消

取消打开或保存文件选择并关闭文件浏览器 Esc 或使用信息编辑器标题中的 返回上一个 。

主要区域¶

导航¶

输入目录

在一个目录单击 LMB 进入该目录。

父目录 Backspace, P

让你到上一层目录。

方向键¶

目录导航也可以通过方向键进行，按下 Alt 和方向键 ：

  • 转到父目录 Alt-Up

  • 上一级目录 Alt-Left

  • 下一个目录 Alt-Right

文件拖放¶

你现在可以简单地将文件从本地文件浏览器拖放到Blender 文件浏览器中。这会将文件浏览器重定位到已删除文件的目录，并选择该文件。

选中项¶

选择

使用 LMB 和 RMB .

选中 / 取消选中全部文件 A

切换选择所有文件。

拖动

拖动 LMB 形如 框选。

方向键¶

也可以通过使用方向键“遍历”来选择/取消选择文件：

  • 只使用方向键，将选中所选方向的下一个文件，并取消选择所有其他文件。

  • 在执行此操作时按住 Shift 不会取消选择任何内容，因此它可以扩展到选择内容，并且允许通过导航到已经选择的文件块（按顺序至少包含两个文件）来取消选择文件。

  • 按住 Shift-Ctrl 进一步选择/取消选择中间的所有文件。

如果未选择文件，方向键导航会根据箭头方向选择目录中的第一个或最后一个文件。

如果你选择一个目录并点击 Return ，你将进入该目录（并突出显示“父”条目会将你带到一个级别）。

文件管理¶

删除文件 Delete, X

删除当前选定的文件。

重命名 Ctrl-LMB

可以用于在文件或目录中给它重命名。
