工具¶

点选¶

参考

模式

所有模式

快捷键

RMB 和 Shift-RMB

最简单的选择形式是使用 RMB 点击物体。要加选物体，按住 Shift-RMB 点击其它想要加选的物体。如果物体在视图中重叠，你可以使用 Alt-RMB 来循环选择（仅 物体模式）。如果你想以这种方式添加其它物体到选择，那么快捷键变为 Shift-Alt-RMB 。要激活已经选择的物体，在其上点击 Shift-RMB。

要 取消 选择一个激活物体，点击 Shift-RMB 一次，如果物体没有激活，点击两次。请注意，这只有在鼠标下没有其它物体的情况下才有效。否则，只是将这些添加到选择。这个bug似乎没有解决方法。

用户还可以使用 Ctrl-RMB 和 Shift-Ctrl-RMB 选择物体点中心点，而非选择其内容。

框选¶

参考

模式

物体模式和编辑模式

菜单

选择 ‣ 框选

快捷键

B

请按 B 激活该工具。使用 框选 工具时按住 LMB 画一个矩形。任何位于该矩形内部的物体都将被选中。如果任何上次激活的物体出现在选择中，它仍将是激活状态。

要取消选择对象，请使用 MMB 或再次点击 框选 同时按住 Shift 或 Alt 。要取消选择，请使用 RMB 。

Hint

框选 会添加先前的选中的物体，所以为了只选择矩形里面的内容，首先用 A 取消全选。

刷选¶

参考

模式

物体模式和编辑模式

菜单

Select ‣ 刷选

快捷键

C

圆圈选择 C 通过按住 LMB 移动带有虚线圆圈的物体来使用。你可以通过圆圈触摸区域来选择任何物体。可以通过滚动 Wheel 或者使用 NumpadPlus 和 NumpadMinus 来动态改变圆的直径，如下图所示。取消选择是基于相同的原则 MMB。要取消选择，请使用 MMB 或者键 Esc 或者 Return 。

套索选择¶

参考

模式

物体模式和编辑模式

快捷键

Ctrl-LMB

套索选择通过在 物体模式 中绘制围绕顶点或物体原点的虚线来使用。要使用这个，按住 Ctrl-LMB 并且简单地绘制你想要选择的点。

套索选择会添加先前的选中的物体。要取消选择，请使用 Shift-Ctrl-LMB。

全选/取消全选¶

参考

模式

所有模式

菜单

选择 ‣ 全选/取消全选, 选择 ‣ 反转

快捷键

A

如果选择了任何内容，则首先取消选择。否则，它会在选择和取消选择每个可见物体之间切换。

命令

选择，取消选择，反转，切换

反转 Ctrl-I

选择所有未被选中的物体，同时取消选择所有已经选定的物体。

随机¶

参考

模式

物体模式和编辑模式

菜单

选择 ‣ 随机

在当前激活层上随机选择基于百分比概率的未选物体。在选择操作时，在 工具架 中可以使用数字选择框。需要注意的是，百分比表示未选中的物体被选中的可能性，而不是将被选中的物体的百分比数量。

按层全选¶

参考

模式

物体模式

菜单

选择 ‣ 按层全选

选择同层所有的可见物体。¶

该选项允许选择属于给定图层的单个物体。所选物体变得可见。

匹配

匹配类型选择。

扩展

启用，物体添加到当前选择，而不是替换当前选择。

层

物体所在的图层。

Tip

物体的选择

相对于使用 Select All by Layer 选项使所需的图层可见，使用 A 可能更有效。该方法还允许取消物体的选择。

扩展/缩减选择¶

参考

模式

物体模式

菜单

选择 ‣ 扩展/缩减选择

快捷键

Ctrl-NumpadPlus, Ctrl-NumpadMinus

它们的目的是基于层次结构的。

扩展选区

将选择扩展到选中项点直系父级与子级。

缩减选区

取消选择父/子关系边界处的物体。

父级

取消当前物体选中状态，并选中其直系父级。

子级

取消当前物体选中状态，并选中其直系子级。

扩展父级

将选择扩展到当前选中物体的直系父级。

扩展子级

Extends the selection to the immediate children of the currently selected objects.

按组选择¶

参考

模式

物体模式

菜单

选择 ‣ 按组

快捷键

Shift-G

有两种方法来组织物体的相互关系。第一个是 父子，第二个是简单 分组。这些关系通过选择相应的父子或组的成员来达到艺术家的优势。 按组选择 使用激活物体作为选择所有其它物体的基础。

子级

选择激活物体的所有层次结构的子级。

直接子级

选择激活物体的所有直接子级。

父级

选择此物体的父级 (如果它有)。

平级

选择与激活物体具有相同父级的物体。这也可以用于选择所有的根级别物体 (没有父级的物体)。

类型

选择与激活物体类型相同的物体。

层

至少有一个共享层的物体。

群组

如果这些物体位于激活物体的某个组中, 物体所在的组将被选择 (以默认主题呈现绿色)。如果激活物体属于多个组, 则会弹出一个列表,从中选择要选择的组。

钩挂

属于激活物体的每个钩挂。

通道层

选择分配给相同的 渲染通道 的物体。

颜色

选择具有相同的 物体颜色 的物体。

插帧集

选择包括在激活中 Keying Set 的物体。

灯光类型

选择匹配的灯光类型。

通道索引

选择匹配的物体通道索引。

选择关联项¶

参考

模式

物体模式

菜单

选择 ‣ 选择关联项

快捷键

Shift-L

选择与激活物体共享一个公共数据块的所有物体。 选择关联项 使用物体对象作为选择所有其它物体的基础。

物体数据

选择每一个链接到相同物体数据的物体，即指定类型（网格，曲线等）和构建（构成元素，如顶点，控制顶点和它们在空间中的位置）的数据块。

材质

选择每一个链接到相同材质数据块的物体。

纹理

选择每一个链接到相同纹理数据块的物体。

组副本

选择使用同一个 组 进行复制的所有物体。

粒子系统

选择使用相同的 粒子系统 的所有物体。

库

选择具有相同的 库 所有物体。

库 (物体数据)

选择具有相同的所有物体 库 ，并限制为 物体数据 。

按名称选择¶

参考

模式

物体模式

菜单

选择 ‣ 按名称

选择名称与给定名称匹配的所有物体。支持的通配符: * 匹配所有内容，？匹配任何单个字符，[abc]匹配“abc”中的字符，并且[！abc]匹配不在“abc”中的任何字符。例如 *house* 与任何包含“house”的名称匹配，而floor* 匹配以“floor”开头的任何名称。

区分大小写

匹配可以选择是否区分大小写。

扩展

当 扩展 复选框被选中时，选择被扩展而不是生成一个新的。
