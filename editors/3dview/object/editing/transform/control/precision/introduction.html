简介¶

参考

模式

物体模式和编辑模式

快捷键

Ctrl 和/或 Shift

在变换期间（如抓取，旋转或缩放）按住 Ctrl 切换到 变换吸附 。当 吸附元素 设置为 增量 时，这允许变换以离散的数量执行。

在变换操作过程中按住 Shift ，将以正常模式的1/10的速度变换物体，从而实现更好更精确的控制。

变换的大小可以在3D视图的左下角的标题栏中查看。在变换过程中释放 Ctrl 或 Shift 将导致移动恢复到正常操作模式。

同时按住 Shift-Ctrl 可以启用精确吸附。该选项将以高精确度吸附约束移动物体。

Note

此页面上描述的吸附行为 仅 适用于当 增量吸附 被选中的情况。

用法¶

使用热键¶

按 G, R 或 S 然后按住 Ctrl, Shift 或 Shift-Ctrl。

使用变换控制器¶

按住 Ctrl, Shift 或 Shift-Ctrl 并点击相应的控制手柄。 然后将鼠标移动到所需的方向。相反的动作也将起作用，即点击控制手柄，然后按住快捷键进行精确控制。

See also

阅读更多关于变换控制器。

Tip

结合其他控制

页面上详细介绍的所有精度控制都可以与 轴向锁定 控制结合使用，并可以使用不同的 轴心点。

吸附¶

抓取 / 移动¶

1 Blender 单位 (默认缩放级别)。¶

对于默认缩放级别的抓取/移动操作，按住 Ctrl 将使你的选择以 1 Blender 单位（1 BU）的增量移动（即在两条浅灰线之间）。现在放大足够看到下一组灰线，将导致按住 Ctrl 以1/10 BU移动。进一步放大，直到下一组灰线变为可见，将导致发生1/100 BU移动等等，直到达到缩放限制。缩小会产生相反的效果，导致移动以10 BU，100 BU等增量发生。

See also

了解更多关于 缩放 的信息。

旋转¶

按住 Ctrl 将导致每次旋转5度。

缩放¶

按住 Ctrl 会以0.1 BU的增量引起大小变化。

Note

吸附模式

请注意，如果启用了 吸附元素 选项，则按住 Ctrl 将导致选择对齐到最近的元素。

了解更多关于 吸附 的信息。

精确¶

在变换期间按住 Shift 允许非常精细的控制, 不依赖固定的增量。 如，鼠标在屏幕上的大幅度移动只会导致选择的物体小幅度的变换。

在旋转模式下, 所选元素将以0.01 度增量旋转。

精确吸附¶

抓取 / 移动¶

对于默认缩放级别的抓取/移动操作，按住 Shift-Ctrl 将使你的选择以1/10的Blender单位增量移动。在任何缩放级别下按住 Shift-Ctrl 将导致变换增量始终为只按住 Ctrl 时增量的十分之一。

旋转¶

按住 Ctrl 将导致每次旋转1度。

缩放变换¶

按住 Shift-Ctrl 将导致大小（尺寸）每次缩放以0.01 BU单位的增量值。
