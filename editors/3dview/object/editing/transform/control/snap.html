吸附¶

在Blender中可以使用两种类型的捕捉操作。第一种类型将你的选择或光标捕捉到给定的点，而在转换（平移，旋转，缩放）期间使用第二种类型，并将你的选择捕捉到场景中的元素。

吸附（捕捉）菜单¶

参考

模式

物体，编辑和姿势模式

菜单

物体/物体类型 ‣ 吸附

快捷键

Shift-S

吸附 菜单 (在 物体模式 和 编辑模式 3D标题栏 物体 ‣ 吸附 和 网格 ‣ 吸附). 这个菜单提供了一些选项来移动光标或你的选择到一个定义的点（光标，选择或网格）。

选中项 -> 栅格点

将当前选定的物体吸附到最近的网格点。

选中项 -> 游标

将当前所选物体中的每一个移动到游标位置。

选中项 -> 游标 (偏移)

将当前选择物体放置在3D光标的位置。如果选择了多个物体，则不会单独移动到游标位置; 而是以3D游标为中心，保持相对距离。

选中项 -> 活动项

将选择移到活动（激活）物体的原点。

游标 -> 选中项

将游标置于当前选择的中心，除非参见下文。

游标 -> 中心点

将游标放置到世界的原点（位置0,0,0）。

游标 -> 栅格点

将游标放在最近的栅格点上。

游标 -> 活动项

将光标置于 激活 （最后选定的）物体的原点。

游标 -> 选中项 选项也受当前 轴心点 的影响。例如:

  • 边界框中心 枢轴点处于活动状态时，游标 -> 选中项 选项会将3D游标捕捉到围绕物体原点的边界框的中心。

  • 当选择 质心点 枢轴点时，游标 -> 选中项 会将3D游标捕捉到物体原点的 质心 。

吸附元素¶

参考

模式

物体，编辑和姿势模式

标题栏

吸附元素

快捷键

Shift-Ctrl-Tab

吸附元素菜单。¶

体积

吸附到鼠标光标下方第一个物体的体积内的区域。与其他选项不同，这一个控制变换元素的深度（即当前视图空间中的Z坐标）。通过切换吸附目标菜单右侧的按钮（见下文），在确定体积中心时，目标物体将被视为一个整体。

面

吸附到网格对象中的面的表面。有用于重心拓扑。

边

吸附到网格物体的边缘。

顶点

捕捉到网格物体的顶点。

增量

捕捉到栅格点。在正交视图中，对齐增量根据缩放级别而变化。

Note

在这种情况下，栅格并不意味着显示可视栅格线。吸附将使用显示的栅格的分辨率，但是所有变换都是相对于初始位置（吸附操作之前）。

吸附目标¶

如果将 顶点，边，面 或 体积 选为吸附元素，则吸附目标选项将变为活动状态。这些决定选择什么部分吸附到目标物体。

活动项

将活动元素（编辑模式中的顶点，物体模式中的物体）移动到目标。

质心

将选择的质心移至目标位置。

中心

将当前变换中心移动到目标。可以使用选中项 -> 游标 (偏移)进行吸附。

邻近

将选择的最近点移动到目标。

多个吸附目标¶

多个吸附目标。¶

一旦用吸附变换一个选择（不只是当按住 Ctrl），您可以按 A 来标记当前的吸附点，然后进行标记尽可能多的其他吸附点，你想和选择物体将被吸附到所有标记点的平均位置。

多次标记一个点将使它位置上更加平均。
