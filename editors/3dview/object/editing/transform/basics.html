基础变换¶

抓取 / 移动¶

参考

模式

物体模式，编辑模式和姿势模式

面板

工具栏 ‣ 工具 ‣ 变换 ‣ 移动

菜单

物体类型 ‣ 变换 ‣ 拖拽/移动

快捷键

G

在物体模式中，抓取/移动选项可以让你平移（移动）物体。平移意味着改变物体的位置。它还可以让你在激活的3D视图的3D空间内平移构成物体的任何元素。

抓取/移动同样可以工作在节点编辑器, 曲线编辑器, UV/图像编辑器, 序列编辑器等。

按下 G 激活“抓取/移动”变换模式。所选物体或元素随后将根据鼠标指针的位置和相机自由移动。

你也可以通过使用 RMB 点击并按住物体来移动。要确认操作，请按 LMB 。

在“抓取/移动”处于激活状态时，X，Y和Z坐标的变化量显示在3D视图编辑器的左下角。

变换显示。¶

Tip

在“物体模式”中移动物体可更改物体的原点。在编辑模式中移动物体的顶点/边/面不会改变物体的原点。

旋转¶

参考

模式

物体模式和编辑模式

面板

工具栏 ‣ 工具 ‣ 变换 ‣ 旋转

菜单

物体/网格/曲线/曲面 ‣ 变换 ‣ 旋转

快捷键

R

旋转也称为旋绕，扭转，轨道，枢纽旋转，围绕或滚动，并且涉及围绕一个或多个轴或者 枢轴点 改变以下元素（顶点，边，面，物体等）的方向。

旋转的角度数将显示在3D视图编辑器的页脚中。

旋转值。¶

轨迹球旋转¶

自由旋转模式。按 R-R 启用轨迹球旋转。

缩放¶

参考

模式

物体模式和编辑模式

面板

工具栏 ‣ 工具 ‣ 变换 ‣ 缩放

菜单

物体/网格/曲线/曲面 ‣ 变换 ‣ 缩放

快捷键

S

缩放意味着改变物体的比例。按 S 将进入 缩放 变换模式，根据鼠标指针的位置将选定元素向内或向外缩放。当鼠标指针从枢轴点移开时，元素的缩放将增加，随着指针向其移动而减小。如果鼠标指针从 Pivot Point 的原始边交叉到另一边，则缩放将继续沿着负方向并翻转元素。

基本的缩放用法。从左到右，面板显示：原始物体，缩小的物体，放大的物体和翻转缩放物体。¶

缩放量将显示在3D视图编辑器的页脚中。

缩放值。¶

常用选项¶

有多种变换元素的方法，其中包括：

  • 键盘快捷键。

  • 标题栏中的菜单或工具架中的“变换”面板。

  • 3D变换控制器 小部件。

  • 在“属性”区域或“物体”选项卡中的 变换面板 。

确认和取消¶

LMB 单击以接受更改变换。这种行为可以通过激活 用户首选项 中的 释放为确认 进行全局更改，以便使用 RMB 拖动表示移动和确认。

要取消变换，请按 RMB 或者 Esc 。这会将物体或元素重置为其原始状态。

See also

使用快捷键组合可以更好地控制变换。请参阅 变换控制 。

编辑蜡笔

ToDo.

缩放纹理空间 Shift-TShift-Alt-T

该复选框用于对 纹理空间 应用变换，而不是在物体或元素本身。仅适用于变换和缩放。该选项还可以通过 Shift-T (移动) 和 Shift-Alt-T (缩放)实现。

松开时确认

显示操作是拖放和释放鼠标操作还是移动和确认操作。

工作流程¶

使用键盘快捷键¶

  1. 使用 RMB 来选择你想要变换的元素。

  2. 点击 G 或者 R 或者 S 一次进入变换模式。

  3. 通过移动鼠标来变换元素。

  4. LMB 点击接受更改。
