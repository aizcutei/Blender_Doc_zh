面副本¶

参考

模式

物体模式

面板

物体 ‣ 副本

菜单

物体 ‣ 生成面副本

Duplication Faces（面副本） 或 DupliFaces（面副本） 是在父物体的每个面上复制物体的功能。解释这个最好的方法之一是通过一个例子来说明。

缩放

根据对应面的大小缩放每个实例。

继承缩放

缩放 ＊面副本* 物体。

生成面副本 工具将关联物体转化为面副本实例。该工具在物体在物体所在位置创建由单个面构成的父级物体(复制器)，然后使用 ＊面副本* 将实例放置在创建面的位置。

可以使用 物体 ‣ 应用 ‣ 副本独立化, Shift-Ctrl-A 从 面副本 实例重新转化为多关联物体。

在此页面 这里 下载用于示例的blend文件。

基本用法¶

在这个例子中，我们将使用一个挤压出“北极”的UV球（棱角球）作为我们的基础物体和立方体作为我们的父级网格。要将棱角球作为子级父链接到立方体，在 物体模式 中，首先 RMB 选择球体，然后 Shift-RMB 选择立方体（顺序非常重要），最后 Ctrl-P 。

一个立方体和一个球体。¶

面副本应用到立方体。¶

接下来，在 物体选项卡 ‣ 副本面板 中，启用 面 。球体的副本复制到立方体的每个面方向。

Note

继承的属性

副本子级的位置，方向和缩放与父级的面相匹配。因此，如果有多个物体作为子级是父链接到立方体的，则链接到立方体上的每个物体都将被复制一次。如果立方体被细分，则每个子级副本数量将和立方体上的面数对应。

父物体和原始物体都在3D视图中显示为可编辑的“模板”，但不会最终渲染。

缩放¶

缩放启用。¶

立方体顶面缩小了。¶

通过为父物体启用 缩放，子物体的比例将根据父物体中每个面的大小进行调整。

因此，通过重新缩放父物体的面，副本物体的大小将相应地改变。

限制/注意事项¶

副本几何图形相对于面的位置取决于子物体相对于基础物体原点的位置。这可能会导致编辑器中出现一些视觉瑕疵，因为原始物体的几何体与副本项重叠或相交。一种解决方法是将副本网格的原点从面的平面移开。

如果子级几何体 不对称，那么面部的方向（由其顶点的顺序决定）可能很重要。从2.70开始，Blender没有工具可以调整面顶点的顺序。

但是，有一个工作流程可以让你控制这个问题。制作一个正方形并启用复制/面，以便在编辑器中看到副本的几何体。如果方向不是你想要的，旋转面部，直到它是你想要的。通常你想在编辑模式下进行旋转，而不是对象模式，但这不是一个硬性规则。

一旦你的方向正确，复制面部并移动你想要的副本。重复这个过程直到你有足够的面。由于这些面互相碰撞是很常见的，所以几何体将会有很多重复的顶点。使用工具面板中的移除重叠点按钮。
