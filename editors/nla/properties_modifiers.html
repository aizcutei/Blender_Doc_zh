属性 & 修改器¶

属性¶

片段属性可以通过NLA属性区域进行访问。

动画数据¶

参考

面板

属性栏 ‣ 动画 ‣ 动画数据

动画数据面板。¶

动作

A 数据块 菜单，允许你编辑动作轨道中显示的动作。另请参阅动作编辑器 Action.

外推动作

采取动作超过片段范围的程度。

保持

影响片段的两侧。

保持前进

只影响剪辑后的区域。

无

都不是。

混合动作

Affects how the property values directly produced by the strip are combined with the result of evaluating the stack below. The bottom-most strip is blended on top of the default values of the properties.

替换

The top strip is linearly blended in with the accumulated result according to influence, completely overwriting it if influence is set to 100%.

相乘，相减，相加

The result of the strip is multiplied, subtracted, or added to the accumulated results, and then blended in according to influence.

\(result = mix(previous, previous (+-*) value, influence)\)

Combine

Depending on the type of each property, one of the following methods is automatically chosen:

Axis/Angle Rotation

\(result = previous + value * influence\)

This results in averaging the axis and adding the amount of rotation.

Quaternion Rotation

Quaternion math is applied to all four channels of the property at once:

\(result = {previous} \times {value} ^ {influence}\)

Proportional (Scale)

\(result = previous * (value / default) ^ {influence}\)

Others

\(result = previous + (value - default) * {influence}\)

This allows layering actions that can also be used as a standalone. Properties keyframed at their default values remain at default.

Note

Since this blending mode is based on using quaternion multiplication for Quaternion Rotation properties, it always drives all four channels during playback, and Insert Single Keyframe is forced to insert all four keys. Other types of channels can still be keyed individually.

动作影响

Amount the active Action contributes to the result of the NLA stack.

活动的轨道¶

参考

面板

属性栏 ‣ 动画 ‣ 活动轨

名称

片段当前所属的轨道的名称。

激活片段¶

Active Strip panel.¶

片段本身的选项。

名称

重命名片段。

类型

根据三种类型的片段，要么是“动作剪辑”，“过渡”，要么是“复合”。

片段范围

片段本身的边界。请注意，这将延长动作的持续时间，它不会导致动作中用更多或更少的关键帧进行播放（请参阅下面的选项）。

外插模式

请参阅上面的 外推动作 。

混合

请参阅上面的 混合动作 。

自动混入 / 混出

从片段重叠处开始创建斜坡。第一片段是完全控制，并且在重叠时间段结束时它线性地斜升，给第二片段完全控制。

淡入

设置表示该片段何时具有完全影响的帧。

淡出

设置该片段完全影响最后一帧。

禁用(静音)

禁用（静音）单个片段（如禁用上面轨道）。导致轨道轮廓显示虚线。

反向

导致此片段完全向后播放。

动作剪辑¶

参考

面板

属性栏 ‣动画 ‣ 活动片段

动作剪辑面板。¶

这表示片段的“物体数据”。很像物体的变换值。

动作

包含在片段中的动作的引用。可以更改为用另一个动作替换当前片段的值。

动作范围

多少动作要使用。

注意：如果选择高于或低于动作的实际关键帧计数的值，则将查阅F曲线外推。可以在曲线编辑器中更改 通道 ‣ 外插模式。

同步长度

导致上面的“开始”和“结束”帧被重置为动作的第一帧和最后一帧。

同步动作长度“现在”

导致上面的“开始”和“结束”帧被重置为动作的第一帧和最后一帧。

回放设定
缩放

拉伸片段，增加 Strip Extents：End Frame 的另一种方法，如上所述。

重复

还扩展了片段，但通过从第一个关键帧循环并前进。

解算¶

参考

面板

属性栏 ‣ 动画 ‣ 解算

解算面板。¶

这决定了片段的影响程度以及经过的时间。

动画影响力

允许改变这个片段的影响程度作为关键帧的值。如果不影响动画，则在重叠期间，片段会线性褪色。

这些可以在 NLA控制曲线 下的摄影表或曲线编辑器中找到，并且看起来像组通道。他们出现在该频道的所有通道/ 函数曲线之前。

动画的片段

与 动画影响力 相同，但带 片段时间 。

循环片段时长

在动作开始和结束之间循环动画时间。
