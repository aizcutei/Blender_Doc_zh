编辑¶

默认情况下，当添加新通道时，曲线编辑器 会将它们设置为 编辑模式 。

许多快捷键与 3D视图 相同，例如：

  • G 移动

  • R 旋转

  • S 缩放

  • B 用于框选/取消选择

当然，你可以在变换过程中通过按下 X 或 Y 来锁定X轴（时间框架）或Y轴（值）的变换。

要精确控制关键帧的位置和值，可以在属性区域的 活动关键帧 中设置值。

插入关键帧¶

参考

快捷键

Ctrl-LMB, Shift-Ctrl-LMB

Ctrl-LMB 将一个关键帧插入到鼠标位置的活动F曲线。新添加的关键帧将被选中，以便快速调整新添加的关键帧。所有先前选择的关键帧都通过使用 Shift-Ctrl-LMB 来保持选择状态。

变换吸附¶

当使用 G, R, S 变换关键帧时，变换可以吸附到增量。

吸附变换为 1.0 Ctrl 。

将变换划分（Divide？）为10 10.0 Shift。

清理通道¶

参考

菜单

关键帧 ‣ 通道

快捷键

X

像 清理关键帧 工具行为，但是如果只剩下一个包含默认属性值的关键帧，并且它没有被任何生成的函数曲线修改器或驱动使用，它也会删除通道本身。

Note

清理工具运行后留下的修改曲线与原始不同，因此在执行函数曲线的自定义编辑之前以及初始关键帧插入之后，可以更好地使用此工具，以便在执行批量关键帧时删除插入的任何不需要的关键帧插入（通过选择全部骨骼并按下：例如 I ）。

平滑¶

参考

菜单

关键帧 ‣ 平滑关键帧

快捷键

Alt-O

还有一个选项可以平滑所选曲线，但要注意：它的算法似乎是将每个关键帧与曲线的平均线性值之间的距离除以2，而不进行任何设置，从而给出相当强的平滑度！请注意，第一个和最后一个帧似乎永远不会被此工具修改。

将声音烘焙成函数曲线¶

参考

菜单

关键帧 ‣ 将声音烘焙成函数曲线

将声音烘焙成函数曲线 工具需要一个声音文件并使用其声波创建动画数据。

最低频率

应用于音频数据的高通滤波器的截止频率。

最高频率

应用于音频数据的低通滤波器的截止频率。

攻击时间

船体曲线计算值指示船体曲线可以上升的速度。值越低，它可以上升越陡峭。

发布时间

船体曲线计算的值，表示船体曲线可以下降的速度。值越低，越容易下降。

阈值

影响船体曲线所需的最小振幅值。

累积

只有船体曲线振幅的正差异被总结出来才能产生输出。

添加

总结了船体曲线的振幅。如果启用 累计，则累计积极和消极差异。

正方形

将输出显示为平方曲线。负值始终为-1，正值为1。

平方阈值

所有低于此阈值的值都会导致0。
