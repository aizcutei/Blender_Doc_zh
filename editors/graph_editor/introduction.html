简介¶

曲线编辑器是主要的动画编辑器。它允许你使用 函数曲线 修改任何属性的动画。

曲线编辑器有两种模式，函数曲线 用于 动作，驱动程序 的 驱动器。两者在功能上都非常相似。

曲线编辑器。¶

曲线视图¶

The curve view allows you to view and edit F-Curves. An F-Curve has several key parts:

Curve

The curve defines the value (Y axis) of the property over time (X axis).

见 函数曲线。

Keyframes

Keyframes are user-defined values on certain frames and are represented by little black squares. These become orange if selected.

见 关键帧 获取更多信息。

Handles

Each keyframe has a handle that helps determine the values of the curve between keyframes. These handles are represented by extruding lines with circular ends and can be selected and modified to change the shape of the curve.

See 控制柄类型 for more information.

A simple curve.¶

See also

请参阅 函数曲线 了解更多信息。

导航¶

与大多数编辑器一样，可以：

平移

单击并拖动（MMB），平移垂直视图（值）或水平视图（时间）。

缩放

用鼠标滚轮放大和缩小（Wheel）。

缩放视图

缩放垂直或水平视图(Ctrl-MMB)。

2D 指针¶

曲线编辑器2D 指针。¶

由称为 时间指针 的绿色垂直线表示当前帧。

如 时间线 那样， 你可以通过按住 LMB 来更改当前帧。

绿色的水平线称为 指针 。这可以通过 视图菜单 或 视图属性 面板禁用它。

时间指针 和 指针 组成 2D指针 。2D指针 主要用于编辑工具。

See also

见曲线编辑器 View Properties。

视图轴¶

对于 动作 X轴代表时间，Y轴代表设置属性的值。

对于 驱动 X轴代表 驱动值，Y轴代表设置属性的值。

根据选定的曲线，这些值具有不同的含义：例如旋转属性以度为单位显示，位置属性以Blender单位显示。请注意 驱动 使用弧度来表示旋转属性。

预览范围¶

设置预览范围 P

Interactively define frame range used for playback. Allows you to define a temporary preview range to use for the Alt-A real-time playback (this is the same thing as the Playback Range option of the Timeline editor header).

清除预览范围 Alt-P

清除预览范围。

自动设置预览范围 Ctrl-Alt-P

自动设置预览范围以播放整个动作或选定的NLA条。

标记菜单¶

Markers are used to denote frames with key points or significant events within an animation. Like with most animation editors, markers are shown at the bottom of the editor.

Markers in animation editor.¶

For descriptions of the different marker tools see Editing Markers.

曲线模式。¶

视图控制¶

视图控制。¶

仅显示已选定（鼠标光标图标）

只包含与所选物体和数据相关的曲线。

显示隐藏（幽灵图标）

包含来自不可见的物体/骨骼的曲线。

仅显示错误（救生圈图标）

只包含被禁用或有错误的曲线和驱动程序。用于调试。

搜索过滤器（放大镜图标） F

仅包含搜索字段中包含关键字的曲线。

Multi-Word (az 图标)

模糊/Multi-Word （多字）名称过滤匹配单词片段/部分单词，而不必匹配所有内容。它基于空白位置细分搜索字符串。例如“lo ro”将过滤所有位置和旋转，而“lc rt”将 不 工作。

过滤类型

按属性类型过滤曲线。

数据块排序（az图标）

物体数据块按字母顺序显示，以便更容易找到它们发生的位置（以及帮助将相关物体的动画以实例保存在NLA中）。

如果你发现你的播放速度受到启用（在场景中处理大量物体象时应该只是一个问题），你可以关闭此功能。

规格化

规格化曲线，使最大或最小点等于1.0或-1.0。

自动

自动重新计算每条曲线编辑的曲线标准化。这对于防止曲线在调整后跳跃很有用。
