简介¶

Blender提供了一系列可以添加到序列中的效果。每个效果将分别在下一页中介绍，但它们都以同样的方式添加和控制的。

要添加一个效果片段，选择一个基础片段（图像，电影或场景）在其上使用 RMB 点击它。对于一些效果，比如交叉过渡效果，你需要把 Shift-RMB 换成第二条重叠片段（这取决于你想要的效果）。然后选择 Add ‣ Effect ，选择你想要的效果。当你这样做时，效果片段将显示在源片段之上。如果它是一个独立的效果，像 Color Generator,它将被放置在帧指示器的位置。

Note

由于大多数带的影响依赖于一个或两个源片段，其帧的位置和持续时间取决于其源片段。因此，你可能无法将其移动; 你必须移动源片段，以影响效果片段。

要使用组合或制作过渡的效果，请选择两个片段，添加效果片段时，它将放置在两个片段上方的通道中。其持续时间将是两条片段之间的重叠最大值。

有一些效果，如 Alpha Over,选择片段的顺序是非常重要的。也可以用一个效果片段作为另一片段的输入或源片段，从而将效果叠加在一起。

Note

唯一的例外是 Color Generator 效果。它不依赖于一个基础片段; 你可以添加和定位它独立于任何其他片段。就像任何片段更改长度那样。

如果你从菜单中选择了错误的效果，你可以随时使用 Change 操作命令进行更换。
