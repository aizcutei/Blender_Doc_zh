清除 & 应用¶

清除¶

参考

模式:物体模式
菜单:物体 ‣ 清空 ‣ 位移 / 缩放 / 旋转 / 原点
快捷键:Alt-G, Alt-S, Alt-R

清除变换只是简单重置变换值。物体位置和旋转值返回到0，并且缩放返回到1。

清除位置 Alt-G
清除（重置）选择物体的位置。这会将选择移回到坐标(0，0，0)。
清除缩放 Alt-S
清除（重置）选择的缩放。这会将选择的物体大小重新调整到创建时的大小。
清除旋转 Alt-R
清除（重置）选择的旋转。这将在每个平面上将选择旋转设置为0度。
Clear Origin
清除 (重置) 子物体原点的偏移量。这将导致子物体移到父级的原点。

选项¶

清除增量
在清除常规变换的同时清除 增量变换。(位于 调整上一步操作 面板)

应用¶

这些操作允许你对选定物体应用多个变换。物体变换坐标转换到物体数据。如果物体具有层次的结构子级, 它也会将这些转换应用于其子级。

应用物体变换¶

参考

模式:物体模式
菜单:物体 ‣ 应用 ‣ 位移 / 旋转 / 缩放 / 旋转 & 缩放
快捷键:Ctrl-A

应用变换值实质上是重置物体的位置、旋转或缩放值，物体仍位于原本位置。物体原点被移动到全局原点，清空旋转，缩放值重置为1。

对于一些简单情况，3D视图或渲染结果是看不出差别的，但是一些修改器和约束可能是基于物体变换的。

Warning

骨架物体

虽然支持将变换应用到骨架，但是这 不会 适用于其姿势位置，动画曲线或约束。需在绑定与动画前使用该工具。

在使用 应用变换 工具时， 调整上一步操作 面板可以使你选择需要应用的变换组合。

选项¶

位置
应用（设置）选择的位置。这将使得Blender认为当前位置在每个平面上等于0，即选择不会移动，当前位置将被认为是“默认位置”。物体原点将设置为实际的（0，0，0）（其中着色的轴线在每个视图中相交）。
旋转
应用（设置）选择的旋转。这将使Blender考虑当前旋转在每个平面上相当于0度，即选择不旋转，当前旋转将被认为是“默认旋转”。
缩放
应用（设置）选择的比例。这将使Blender考虑当前比例在每个平面上等于0，即选择不会缩放，当前比例将被认为是“默认比例”。
旋转和缩放
应用（设置）选择的旋转和缩放。同时做上述两个应用的命令。
应用属性
根据应用的变换来修改类似于曲线顶点半径，字体大小和骨骼蒙皮之类的属性。

变换为增量¶

参考

模式:物体模式
菜单:物体 ‣ 应用 ‣

将"常规"物体变换转化为 增量变换，在现有增量变换量叠加。

  • 位置增量
  • 旋转增量
  • 缩放增量
所有变换为增量
将所有 "正常" 变换为增量变换。
动画变换为增量
将“正常”变换动画（对平移，缩放和旋转值所做的动画）转换为增量变换。

选项¶

重置值
在传递至增量变换后清空 "常规"变换。

可视变换¶

参考

模式:物体模式
菜单:物体 ‣ 应用 ‣ 可视变换
快捷键:Ctrl-A

应用（设置）约束结果并将其应用返回物体的位置，旋转和缩放。

可视几何 -> 网格¶

参考

模式:物体模式
菜单:物体‣ 应用 ‣ 可视几何到网格
快捷键:Ctrl-A

将所有选定对象（修改器，形状键，挂钩等）的可视状态应用于物体数据。这是将所有物体数据冻结到静态网格中的一种方法，也可以将非网格类型转换为网格。

详情见 转换为 网格。

实例独立化¶

参考

模式:物体模式
菜单:物体 ‣ 应用 ‣ 实例独立化
快捷键:Shift-Ctrl-A

Make Instances Real 实例独立化未每个被选择的 实例 生成一个新物体，并移除它们身上所有的直接实例。

最后，每个实例都成为了真正的物体。

Warning

此选项同时作用于直接（来自米顶点或面...）和间接（来自粒子系统...）的实例化。注意如果你直接在一些拥有大量实例的场景（例如粒子系统）中使用，Blender运行速度会迅速下降，毕竟同场景物体太多会很吃性能。

选项¶

默认情况下，新物体会加入到同一个包含了他们的实例器的集合中，而不保留任何层级关系。此行为可以在下列选项中更改。

父级

如果未设定 保留层级 ，将所有生成的物体父级关联到之前的实例器。

此外，将所有生成的（未被父级关联的）物体父级关联到各自的实例器，或者匹配新拷贝（这对于防止重复循环实例化很关键，见下文）

保持层级
在新生成的物体中保留内置的层级（例如父子关系）。

Tip

通常，为了从实例化物体中获得一个尽可能一致的新层级，最好同时启用这些选项。

Note

在循环实例化（实例器实例化其它实例器物体）保留关系目前仅适用于某些方面。

简单场景（例如一个空物体实例化一个包含了其它集合的实例的实例）通常管用，但一些更复杂的场景若不减少实例化层级的话将会不起作用。
