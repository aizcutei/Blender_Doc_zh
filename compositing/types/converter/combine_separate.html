合并/分离节点¶

这类节点本质上都是做同一件事情：

  • 分离：将一幅图像的颜色通道分离成单独颜色通道。

  • Combine: Re/combine an image from its composite color channels.

此类节点可以独立操控单个颜色通道。在不同的 color space 中每种类型之间是有差异的。

在合成和相关的贴图中都支持Alpha通道。在贴图的中只允许使用RGB色彩空间。在Blender内部着色器节点的HSV或Cycles 着色器都提供了额外的可以 合并/分离 矢量（XYZ）的节点。

The Combine nodes could also be used to input single color values. For RGBA and HSVA color spaces it is recommended to use the RGB节点. Some common operations could easier be executed with the 颜色类节点.

范例¶

模糊Alpha通道举例。¶

第一个范例中，首先模糊Alpha通道，然后和其他颜色进行合并。当放置在场景中，通道的边缘会发生颜色融合，而不是一个硬朗的边缘。这个可以称作三维场景中的抗锯齿功能。利用这个节点设置，可以添加有动作的CG元素并移除硬表面的硬边缘。运动物体边缘缩放会产生物体的相位进出，称作“脱相”时间旅行同步特效。

属性¶

模式

不同的色域，应对不同的投放环境。ITU 601（SDTV色彩空间，标清广播色域）, ITU 709（HDTV色彩空间 高清广播色域）, Jpeg（图像压缩标准）
