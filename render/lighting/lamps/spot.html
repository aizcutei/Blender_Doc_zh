Spot¶

Spot(聚光灯) 从圆锥体的顶端向给定方向发出锥形光束。

Spot 灯是最复杂的灯光物体，实际上，在很长一段时间内，它是最常用的，因为它是唯一能够投射阴影的这一事实。如今，将光线追踪集成到Blender的内部渲染引擎中，所有灯都可以投射阴影（除了 Hemi半球）。即便如此，Spot 灯的阴影缓冲渲染要比光线跟踪阴影快得多，特别是在模糊/柔化时，并且聚光灯还提供其他功能，如“体积”光晕。

灯光选项¶

距离、能量和颜色

这些设置对大多数类型的灯都是通用的，说明在 Light Properties 。

阴影¶

聚光灯形状¶

更改此选项会更改3D视图中显示的聚光灯外观。¶

尺寸

Spot 外锥的大小，主要控制 Spot 光线覆盖的圆形区域。实际上，此滑块控制顶部灯光圆锥的角度，可以在（1.0到180.0）之间。

更改聚光灯 SIZE 选项。¶
混合

Blend 滑块控制 Spot 的内锥。 Blend 值可以在（0.0到1.0）之间。该值是成比例的，表示内锥体应在外锥体内占据的空间量 尺寸。

内锥边界线表示来自 Spot 的光开始模糊/柔化的点; 在此之前它的光线将大部分是全强度。 Blend 的值越大，聚光灯的边缘越模糊/柔和，内锥体的圆形区域越小（因为它开始模糊/柔化更早）。

要使 Spot 具有更锐利的衰减率并因此减少模糊/柔和边缘，请减小 Blend 的值。将 Blend 设置为0.0会产生非常清晰的聚光灯边缘，光线和阴影之间没有任何过渡。

Spot 灯光的衰减率是 Blend 和 Size 值之间的比率; 当两者之间的圆形间隙变大，Blend 和 Size 之间的光线逐渐消失。

Blend 和 Size 仅控制 Spot 光锥的光圈和柔和度（“径向”衰减）;它们不能控制如下所示阴影的柔和度。
