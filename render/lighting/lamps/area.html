Area¶

Area（区域/面光） 灯模拟来自表面（或类似表面）发射的光。例如，电视屏幕、办公室霓虹灯、窗户或多云的天空只是几种类型的区域灯。区域灯通过沿用户定义的网格对灯进行采样，产生具有软边界的阴影。这与点状的人造光形成了直接的对比，后者产生了尖锐的边界。

灯光选项¶

距离、能量和颜色

这些设置对大多数类型的灯都是通用的，说明在 灯光属性 。

注意，距离 设置对 区域 灯比其他灯更为敏感和重要；通常，距离 范围内的任何物体都会曝光过度。为获得最佳效果，请将 距离 设置为刚好低于要照亮的问题的距离。

阴影¶

区域灯形状¶

区域灯的形状可以设置为 正方形 或 矩形。

正方形选项。¶

矩形选项。¶
正方形/矩形

从正方形或矩形区域发射光。

Size / Size X / Size Y

正方形 或 矩形 的尺寸。

Note

形状提示

为你的 区域 灯光选择合适的形状将增强场景的可信度。例如，你可能有一个室内场景，希望模拟光线从窗口进入。你可以将 矩形 区域灯放置在窗口（垂直）或，其比例适合 尺寸X 和 尺寸Y 的霓虹灯（水平）中。对于模拟电视屏幕发出的光，在大多数情况下，垂直的 正方形 区域灯会更好。
