材质面板¶

可以在 materials tab ‣ materials panel 中将材质链接到物体和物体数据。在这里，你可以管理如何将材质链接到物体、网格等，并激活材质以在其它面板中进行编辑。

材质面板。¶

材质插槽¶

活动的材质

物体的材质槽在 List View 中显示。

Specials

复制并粘贴选定的材质槽。

多重材质¶

网格可以处理具有多个材质的情况。可以按每个面映射材质，如 多重材质 页面中所述。在编辑模式下，将显示以下工具：

分配指定

将材质槽中的选定材质指定给选定顶点。

选择

选择指定给选定材质槽的顶点。

取消选择

取消选择指定给选定材质槽的顶点。

材质命名和链接¶

材质

所选材质槽的 Data-Block Menu 。

节点

切换 将此材质指定为材质节点设置，而不是使用“材质/渐变/着色器”设置。

数据块链接

指定材质是链接到物体还是链接到物体数据。

链接选择器有两个选项，数据和物体。这两个菜单选项确定材质是链接到物体还是数据（例如网格或曲线）。 “数据”菜单选项确定此材质将链接到网格的数据块，该数据块链接到物体的数据块。 “物体”菜单选项确定材质将直接链接到物体的数据块。

这当然会有后果。例如，不同的物体可以共享同一个网格数据块。由于此数据块定义了物体的形状，因此编辑模式中的任何更改都将反映在所有这些物体上。此外，链接到该网格数据块的任何内容都将由共享该网格的每个物体共享。因此，如果材质链接到网格，每个物体都将共享它。

另一方面，如果材质直接链接到对物体数据块，则物体可以具有不同的材质，并且仍然共享相同的网格。

也就是说：如果连接到物体，则可以使用不同的材质创建同一物体数据的多个实例。如果链接到网格数据，则不能。有关详细信息， Data System for more information.

材质类型¶

在编辑模式下添加材质。这些切换告诉Blender这种材质适合到渲染管道，以及材质的哪些方面要渲染。

表面

Render object as a surface.

体积

Render object as a volume.
