标记¶

标记用于对动画中的关键点或重大事件所在帧进行标记。比如，这可能是角色动画开始，相机改变位置，或开门的时间点。可以指定标记名称，这样扫一眼就可以快速甄别。在Blender的多个编辑器中都可以使用该工具。

Note

不像关键帧，标记可以放置在实数数值的帧数处，比如在 2.5帧 处。

标记可以在下面的编辑器中被创建和编辑：

  • 曲线编辑器
  • 动画摄影表
  • NLA编辑器
  • Video Sequence Editor
  • 时间线

Note

如果在某个窗口中创建了一个标记，对于同样支持该标记的其他窗口，也会被显示出来。

类型¶

Besides standard markers, pose markers are another type of markers, which are specific to armatures and shape keys. They are used to denote poses in the Action Editor mode and Shape Keys Editor of Dope Sheet.

可视化¶

标准¶

Most of the editors visualize markers as small white triangles in a separate row at their bottom, empty if unselected or filled if selected.

If they have a name, this is shown to their right in white.

Vertical dashed lines can be enabled via the View ‣ Show Marker Lines menu option.

3D Viewport¶

The 3D Viewport does not allow you to create, edit or remove markers, but it shows their name in the Object Info in the upper left corner, when on their frame.

姿态标记¶

姿态标记在Dope Sheet中显示为菱形的图标。在非线性编辑器（NLA 编辑器）中，姿态标识符显示为红色虚线。

添加标识符¶

参考

模式:所有模式
菜单:标记 ‣ 添加标记
快捷键:M

最简单的添加标记的方式是，移动到所需的帧，按下 M 。

Hint

Markers can also be added during playback.

姿态标记¶

If Show Pose Markers is checked, a pose marker and a new pose in the Pose Library are added.

选择标记¶

参考

模式:所有模式
快捷键:LMB

Click LMB on the marker's triangle to select it. Use Shift-LMB to select multiple markers.

In the Graph Editor, Dope Sheet, NLA Editor, Timeline, and Video Sequence Editor, you can also select all markers with A while hovering the mouse over the marker row, and apply selection tools on them like Box Select, etc. (as usual, LMB to select, RMB to deselect). The corresponding options are found in the Select menu of these editors.

编辑¶

复制标记¶

参考

模式:所有模式
菜单::menuselection:标记 --> 复制标记`
快捷键:Shift-D

You can duplicate the selected markers by pressing Shift-D. Once duplicated, the new ones are automatically placed in select mode, so you can move them to the desired location.

Note

请注意，标记的复制和其他blender数据的复制不同，其名字后面不会被添加".001"等字样。

复制标记到场景中¶

参考

模式:所有模式
菜单:标记 ‣ 将标记复制到场景...

Duplicates the selected markers into another scene.

删除标记¶

参考

模式:所有模式
菜单:标记 ‣ 删除标记
快捷键:X

要删除选择到的标记只需要简单的按下 Shift-X (或者 X 在时间线或音频窗口), 点击 LMB 以确认弹出菜单的选项。

创建标记¶

参考

模式:所有模式
菜单:标记 ‣ 重命名标记
快捷键:Ctrl-M

你在你的时间线中添加一大堆标记却标好名字那肯定也没有什么卵用。选择一个标记，按下[CTRL]+[M]，取个好名字，然后OK。

Move Marker¶

参考

模式:所有模式
菜单:Marker ‣ Move Marker
快捷键:G

Once you have one or more markers selected, press G, while hovering with the mouse over the marker bar, to move them, and confirm the move with LMB or Return (as usual, cancel the move with RMB, or Esc). Or drag them with the LMB.

By default, you move the markers in one-frame steps, but if you hold Ctrl, the markers will move in steps corresponding to one second (according to the scene's FPS).

显示姿态标记¶

参考

模式:todo <2.8
编辑器:动作编辑器和形变键编辑器
菜单:标记 ‣ 显示姿态标记

Pose markers are only shown and editable in Action editor or Shape Keys editor by enabling the Marker ‣ Show Pose Markers checkbox.

设置标记为局部¶

参考

模式:所有模式
菜单:Marker ‣ Make Markers Local

通过 Marker ‣ Make Markers Local 可以将标准的标记转成姿态标记。注意原来的标记将被删除。如果你想保留它，转换前制作一个副本。

跳到下一个或之前的标记¶

参考

模式:所有模式
菜单:Marker ‣ Jump to Next/Previous Marker

Moves the playhead to the next/previous marker relative to the current playhead position.

绑定相机到标记上¶

参考

编辑器:Timeline
菜单:Marker ‣ Bind Camera to Markers
快捷键:Ctrl-B

Bind Camera to Markers is a special operator only available in the Timeline. The operator allows markers to be used to set the active object as the active camera.

To use this operator, select the object to become the active camera and select a marker to bind the active camera to. If no marker is selected when the operator is preformed, a marker will be added. When an object is bound to a marker, the marker will be renamed to the name of the active object. These markers also have a camera icon next to the left of the name to easily distinguish them from other informative markers

These markers can be moved to change the frame at which the active camera is changed to the object the marker is bound to.
