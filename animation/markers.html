标记¶

标记用于对动画中的关键点或重大事件所在帧进行标记。比如，这可能是角色动画开始，相机改变位置，或开门的时间点。可以指定标记名称，这样扫一眼就可以快速甄别。在Blender的多个编辑器中都可以使用该工具。

Note

不像关键帧，标记可以放置在实数数值的帧数处，比如在 2.5帧 处。

标记可以在下面的编辑器中被创建和编辑：

  • 曲线编辑器

  • 动画摄影表

  • NLA编辑器

  • 序列编辑器

  • 时间线

Note

如果在某个窗口中创建了一个标记，对于同样支持该标记的其他窗口，也会被显示出来。

类型¶

姿态标记 是另一种类型的标记，专用于骨架（armatures）和形变键（shape keys）。它们在动作编辑器（Action Editor）中用于表示姿态，在形状键编辑器中表示动画摄影表。

可视化¶

标准¶

标记：小但很有用。¶

许多编辑器窗口都将标记显示成同一种样式：一个放在底部的小三角，选定是黄色的，没选定则是白色的。

如果标记有名字，则会显示在右边，选定时呈现白色。

序列编辑器¶

序列编辑器的标记。¶

视频序列编辑器 会为每个标记添加一个虚线在其对应的位置（灰色表示未选定，白色表示选定）。

3D 视图¶

3D窗口中显示的标记。¶

视窗本身不允许你添加/删除/移动标记，它只会在处于其所在帧时在当前激活物体名字的旁边显示为其名字（参见3D View中的标记）。

姿态标记¶

动作标记器中的姿态标记。¶

姿态标记在Dope Sheet中显示为菱形的图标。在非线性编辑器（NLA 编辑器）中，姿态标识符显示为红色虚线。

添加标识符¶

参考

模式

所有模式

菜单

标记 ‣ 添加标记

快捷键

M 或 Ctrl-Alt-M

最简单的添加标记的方式是，移动到所需的帧，按下 M 。

Hint

标记也可以在重放时添加。

姿态标记¶

如果 显示姿态标记 被选中，一个姿态标记和一个新的姿态在 Pose Library 中会被添加。

选择标记¶

参考

模式

所有模式

快捷键

RMB

在标记上单击右键进行选择。按住[SHIFT]再点击则可以多选。

In the Graph Editor, Dope Sheet, NLA Editor, and Video Sequence Editor, you can also select all markers with Ctrl-A, and apply selection tools on them like Border Select etc. (as usual, LMB to select, RMB to deselect). The corresponding options are found in the Select menu of these editors.

在时间线中，你可以按 A 选择所有标记，使用 B 来框选。

编辑¶

复制标记¶

参考

模式

所有模式

菜单

:menuselection:标记 --> 复制标记`

快捷键

Shift-D

你可以用 Shift-D 复制选择的标记的副本。一旦复制，新的标记自动被设置为抓取模式，以便你可以把它们移动到所需的位置。

Note

请注意，标记的复制和其他blender数据的复制不同，其名字后面不会被添加".001"等字样。

复制标记到场景中¶

参考

模式

所有模式

菜单

标记 ‣ 将标记复制到场景...

Duplicates the selected markers into another scene.

删除标记¶

参考

模式

所有模式

菜单

标记 ‣ 删除标记

快捷键

X

要删除选择到的标记只需要简单的按下 Shift-X (或者 X 在时间线或音频窗口), 点击 LMB 以确认弹出菜单的选项。

创建标记¶

参考

模式

所有模式

菜单

标记 ‣ 重命名标记

快捷键

Ctrl-M

你在你的时间线中添加一大堆标记却标好名字那肯定也没有什么卵用。选择一个标记，按下[CTRL]+[M]，取个好名字，然后OK。

抓取/移动标记¶

参考

模式

所有模式

菜单

标记 ‣ 拖拽/移动标记

快捷键

G

一旦你有一个或多个标记被选定，按 G ， 同时把鼠标放在标记上移动选择的物体（hover），并通过单击左键确定，或者按 Return 键 （像以往一样，取消按鼠标右键，或者 Esc）。 或者用鼠标右键拖拽它们。

默认状态下，你抓取标记算一帧的步长，但当你按住 Ctrl ， 标记将移动对应的秒数 （根据场景的 FPS）。

显示姿态标记¶

参考

模式

动作编辑器和形变键编辑器

菜单

标记 ‣ 显示姿态标记

仅仅姿态标记在动作编辑器或形态键编辑器中被显示和可编辑，可通过勾选 Marker ‣ Show Pose Markers 来实现。

设置标记为局部¶

参考

模式

所有模式

菜单

Marker ‣ Make Markers Local

通过 Marker ‣ Make Markers Local 可以将标准的标记转成姿态标记。注意原来的标记将被删除。如果你想保留它，转换前制作一个副本。

跳到下一个或之前的标记¶

参考

模式

所有模式

菜单

Marker ‣ Jump to Next/Previous Marker

将时间光标移动到相对于当前时间光标位置的下一个或之前的标记。

绑定相机到标记上¶

参考

编辑器

时间轴编辑器

菜单

View ‣ Bind Camera to Marker

快捷键

Ctrl-B

同时选中相机和标记，可以通过时间轴上的操作符 Bind Camera to Markers 交换相机。

当相机为active时，上面的三角形将呈现阴影状。

工作流¶

首先，在时间轴上，添加一组标记用于切换相机，按 M 添加标记，Ctrl-M 重命名，复制的标记需要保持相同的名字。

  1. 在3D View中，选择想用切换到的相机的标记。

  2. 在时间轴上，选择相应的标记以切换到相机上。

  3. 在时间轴上，按 Ctrl-B 绑定相机到标记上。
