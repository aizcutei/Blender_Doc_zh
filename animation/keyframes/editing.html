编辑¶

插入关键帧¶

参考

模式:物体模式
菜单:物体 ‣ 动画 ‣ 插入关键帧...
快捷键:I

有多种方法可以插入关键帧。例如:

  • 在3D视图，按下 I 将弹出菜单，用于选择需要插入的属性。
  • 鼠标在属性值上方停留，并按下 I ，或者在 RMB 单击属性值弹出的上下文菜单中选择 插入关键帧。

自动关键帧¶

时间轴上的自动关键帧。

Auto Keyframe is the record button in the Timeline header. Auto Keyframe adds keyframes automatically to the set frame if the value for transform type properties changes.

请参考 时间轴关键帧控制。

删除关键帧¶

参考

模式:物体模式
菜单:物体 ‣ 动画 ‣ 删除关键帧...
快捷键:Alt-I

有几种删除关键帧的方法:

  • 在3D视图，按下 Alt-I 删除所选物体在当前帧下的关键属性值。
  • 当鼠标停某一属性值上时，按 Alt-I。
  • 在属性值上 RMB 单击，并选择 删除关键帧。

清除关键帧¶

参考

模式:物体模式
菜单:物体 ‣ 动画 ‣ 清除关键帧...

移除选中物体上的所有关键帧。

编辑关键帧¶

可以在两种编辑器中编辑关键帧: 曲线编辑器 或者 动画摄影表。

例子¶

关键帧动画¶

该例子演示了如何使一个立方体产生位置，姿态，和尺寸的动画。

  1. 首先，在 时间轴 上，或其它动画编辑器上，设置当前帧数为1。
  2. 使 立方体 处于 物体模式 ，在3D视图按下 I。
  3. 在 插入关键帧菜单 中选择 位置+旋转+缩放。它将记录立方体在第1帧的位置，姿态，和尺寸。
  4. 设置帧数为100。
  5. Use Move G, Rotate R, Scale S, to transform the cube.
  6. 在3D视图中按 I ，在 插入关键帧菜单 中选择 位置+旋转+缩放 。

可以按下 Alt-A 来预览所制作的动画。
